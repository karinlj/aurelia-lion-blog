# `lion-blog`
## Description

A blog for exploring the Aurelia js CLI.   

![Sreenshot](/static/img/screenshot.jpg?raw=true "Sreenshot")

## Installation

To install

```npm install```

## Run dev app

Run `npm start`, then open `http://localhost:8080`

## To run
```npm run build```

## Built With

* aurelia.js
* Less
* Bootstrap 4
